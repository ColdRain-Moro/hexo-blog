2022年度总结

从今年开始正儿八经的写年度总结

之前其实有想过写一些话，但想想还是算了，只总结我个人技术上的进步

其实仔细想想，2022还是做了一些事的，但其实我不是很满意。OS，编译原理，计网，这些重要的东西都是浅尝则止，希望今年能在这些方面有所突破。2022年的后半年学的东西其实都离 android 开发比较远，不过我觉得这样也好，遵循自己的兴趣学点自己领域以外的新知识也挺好的。况且我学的知识并不是完全对我的未来没有帮助，毕竟我有个做 android 架构的长远目标。

做了

  • 成为了红岩网校的一名干事

    明明去年学长带我的时候都历历在目，今年居然轮到我来带学弟学妹了，这感觉真是奇妙

  • rust 入门

    应该算入门了吧（？）

  • 成为 linux 用户

    主力系统已经换成了 Manjaro Linux 用了半年了，虽说滚挂了好几次，但用起来很舒服

    现在 Linux 的基本操作也算比较熟悉了

  • 给自己的小米11 root 了，玩起了 Hook

    安卓开发者怎么能没有一台 root 过的手机 （

  • 学了点前端（但不多）

    会点前端还是很不错的，前端的技术确实方便整活

  • Kotlin 协程源码解析

    站在蔷神的肩膀上（

    顺便简单分析了js的 async await，异步编程这一块应该算吃透了

  • NIO 原理 / Netty 学习

    NIO 结果弄了半天是靠系统内核支持的多路复用，实际上 java 层的源码也没什么非常巧妙的地方

    Netty 确实是个不错的网络框架

  • 浅学了 KSP (Kotlin Symbol Processor) ， KCP (Kotlin Compiler Plugin)

    确实对这种编译期的东西比较感兴趣

    之后学了编译原理再深入研究

  • etc.

没做到的

  • 学 OS

    在学，不过学着有点吃力，今年学不完了

  • 学编译原理 / 看 Javac 源码

    年末了才开始，决定先把这个学了再继续 OS

  • 学计网

    这个是真的一点没学，不过现在事情确实有点多，计网就先搁一搁吧

  • 写个树洞

    后端其实已经写好了，不过我其实想构思一个去中心化的分布式树洞，但一直没想出来该怎么改

    android 端和前端的进度就暂时搁置了 （咕咕咕）

  • 学 Flutter

    说实话不是很感兴趣了，现在比较想学 native 的东西

    抱歉，在想 rust 的事

  • 学 Gradle

    好吧，其实我还是有点好高骛远了。Gradle 其实才是我现阶段应该琢磨的东西，但我没琢磨多少就去冲 OS 了。

2023的野望

  • 学完老三样 编译原理/OS/计网

    不说学透吧，至少大致要了解，并且能在实践中活用

  • 琢磨 Android NDK 开发

    既然学了 OS，那就顺便把 NDK 也一起学了吧

  • 刷 Arch Linux

    这个其实早就可以着手开始刷了，但重装了系统配乱七八遭的东西还挺麻烦的，但今年一定要抽时间做了

  • 刷 LineageOS

    其实继续 MIUI 也没什么，但不觉得类原生很酷很 Geek 吗，很符合我对 android 开发者的幻想

  • 学 LearningOS ，用 rust 搓一个系统

    这太酷了，很符合我对 OS 开发者的幻想

  • 深入学 Rust

    Rust 太深奥了，我需要在语言层面上投入更多

  • 学 Gradle

    其实我觉得对于 android 开发来说，会 Gradle 足以让你成为团队的核心
