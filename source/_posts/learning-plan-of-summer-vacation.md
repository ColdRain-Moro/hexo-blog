# 2022暑假学习计划

优先度按照顺序来

- JVM
- Jetpack Compose
- 自定义View
- React/Vue/Electron
- Flutter
- Go Web / Java NIO / Java Web

没必要纠结Compose for Desktop / Compose for MultiPlatform，目前我认为桌面应用还是Electron来得快些。Electron有Web前端的生态，而Compose现在生态属实不太行。并且我认为我这段时间前端写得确实挺少，迫切的需要练习，如果开发桌面应用也使用前端的技术的话也能有更多机会磨练前端技术。
